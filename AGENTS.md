# AGENTS

## ひとこと箱 v0.1 の前提

- `quietMode` はユーザーが切り替える機能ではなく、プロダクトの前提です。
- ホーム画面にメンバー切替、共有設定、visibility 設定を出さないでください。
- v0 では「誰が書いたか」を表側 UI に出しません。
- 表側は入れるだけ、裏側でまとめます。
- 新しい設定項目を追加するときは、通常画面のノイズが増えないかを先に確認してください。

## 表側と裏側

- ホームは入力、写真添付、音声入力、保存、あとで見る導線に絞ります。
- review は静かな振り返りに留め、最大 3 件の思想を守ります。
- developer や将来の lab は、Analysis、DeveloperNote、Codex 用指示文、JSON 管理などの裏側情報を置く場所です。
- 通常ユーザーに分類、共有、設定、分析理解を求めないでください。
